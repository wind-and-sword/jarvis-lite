# Jarvis Lite 桌面安装包三阶段计划

> 日期：2026-05-20
> 执行者：Codex

## 目标

把当前可运行的 PySide6 桌面助手推进到可生成 Windows 可分发安装产物的状态。

## 阶段划分

### 阶段 1：桌面体验收口

补应用图标和一致的窗口/托盘身份。该阶段只处理体验收口，不做打包。

### 阶段 2：打包前准备

补冻结运行时目录、PyInstaller 命令生成、打包入口脚本和打包依赖声明。该阶段让项目具备可重复打包条件，但不生成安装器。

### 阶段 3：Windows 安装产物

生成 PyInstaller 桌面 exe，并用 Windows IExpress 生成安装器。二进制产物输出到项目外 `../jarvis-lite-dist/`，不提交 Git。

## 交付规则

- 三个阶段分别提交、分别 push。
- 每个阶段都有独立 `word/` 设计文档和进度文档。
- `.codex/` 继续保存 Codex 本地执行记录，不作为 Git 提交内容。
- 摄像头、麦克风和真实语音识别继续暂缓，不作为安装包阻塞项。
