# 最近建议状态展示第一版计划

> 日期：2026-05-22
> 执行者：Codex

## 目标

让“查看最近上下文”不仅展示最近资料、目录和搜索结果，也展示最近建议列表和当前待确认建议命令。

## 设计

- `_recent_context_status()` 增加 `has_advice_suggestions` 和 `has_pending_advice_command`。
- 有最近建议时展示 `最近建议：N 条`，并列出编号建议文本。
- 有待确认命令时展示 `待确认建议命令：/xxx`；否则展示 `待确认建议命令：无`。
- 空状态判断纳入最近建议和 pending 命令。
- `_remember_recent_advice_suggestions()` 写入新建议时清空旧 pending 命令，避免确认过期建议。

## 验收

- 生成建议后，“查看最近上下文”显示最近建议数量和第一条建议。
- 准备执行建议后，“查看最近上下文”显示待确认建议命令。
- 新建 Agent 恢复最近建议后，“最近上下文状态”也能显示最近建议，但 pending 命令为无。
