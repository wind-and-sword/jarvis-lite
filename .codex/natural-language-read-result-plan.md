# 查看最近搜索结果计划

> 日期：2026-05-21
> 执行者：Codex

## 上下文扫描

- 最近搜索结果编号选择已经保存最近一次问答返回的资料路径列表。
- `/read 文件名` 已通过 `ToolRegistry.read_data_file` 读取 data 文件内容，并限制路径在 `data/` 目录内。
- 当前用户只能给第 N 条结果打标签，不能直接查看第 N 条结果对应的完整资料内容。

## 范围

- 支持问答后说“查看第二条结果”或“查看第2条结果”。
- 返回对应 data 文件路径和文件内容。
- 没有最近搜索结果时给出明确提示。
- 序号超出范围时复用当前结果数量提示。

## 不做

- 不启动外部应用，不打开资源管理器。
- 不跨会话持久化最近搜索结果。
- 不改变知识库检索排序、过滤和回答格式。

## 验收

- RED：问答返回两条资料后，“查看第二条结果”先无法读取第二条资料。
- RED：没有最近搜索结果时，“查看第二条结果”先没有专门提示。
- GREEN：第二条结果可被读取，输出包含 `data/runtime.md` 和文件内容。
- 全量 `unittest`、桌面 smoke、`git diff --check` 通过。
