# v6 InnerBrain 收尾执行计划

> 日期：2026-05-28
> 执行者：Codex

## 目标

把用户已确认的“InnerBrain 本地内脑 + SearchRouter 联网搜索 + LLM 外脑总结”方案继续推进到下一版可安装测试包，避免继续依赖正则作为自然语言主识别路径。

## 实施步骤

1. 联网搜索后续动作：
   - 新增按编号查看/打开最近联网搜索来源的命令与自然语言入口。
   - 新增基于最近联网搜索来源的 LLM 比较入口。
   - 新增保存最近联网搜索摘要到 `word/`，以及导入同一摘要到 `data/` 的入口。
2. 桌面快捷方式删除迁移：
   - 扩展 seed 样本和槽位抽取，覆盖“桌面快捷方式删掉 X”“把桌面快捷方式 X 删除”等表达。
   - 保持只处理明确点名的 `.lnk` 快捷方式。
3. 澄清体验：
   - 中置信或缺槽时给出可执行补全方式，例如 `/inner-brain-label`、`/inner-brain-teach` 和目标命令示例。
4. 文档与版本：
   - 更新 README、项目计划和 v6 方案文档。
   - 版本提升到 `0.1.5`，构建新的 Windows 安装包。
5. 验证：
   - 每个行为先写失败测试，再实现。
   - 运行定向测试、完整 unittest、桌面 smoke、打包后 exe smoke 和安装脚本版本检查。
