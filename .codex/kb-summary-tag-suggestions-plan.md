# 知识库摘要标签建议计划

> 日期：2026-05-25
> 执行者：Codex

## 目标

让 `/kb-summary` 在展示标签分组后，继续给出可直接复制的按标签提问建议，减少用户从摘要到问答的操作成本。

## 实现设计

- 保留现有通用提示：读取第一份资料、给第一份资料打标签、`/ask 关键词`。
- 当知识库中存在标签时，额外追加 `按标签提问：/ask 标签A；/ask 标签B`。
- 标签去重后按名称排序，最多展示 3 个，避免提示过长。
- 空知识库和无标签知识库保持现有输出，不强行生成标签建议。

## 验收标准

- Agent 测试覆盖 tagged `/kb-summary` 输出按标签 `/ask` 建议。
- 现有 `/kb-summary` 编号后续操作和最近资料上下文不变。
- 全量测试、桌面 smoke 和 `git diff --check` 通过。
