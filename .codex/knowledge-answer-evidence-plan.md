# 知识库问答证据增强计划

> 日期：2026-05-22  
> 执行者：Codex

## 目标

让 `/ask` 和普通问题命中知识库时，回答不仅展示来源和命中文本，还说明命中原因，并给出可继续操作。

## 范围

- 输出每条命中的关键词匹配分数。
- 回答末尾提示围绕第一条结果可以继续“查看第一条结果”“给这个结果打标签 标签”或 `/read 文件名`。
- 不改变搜索排序、打分算法、命中数量、弱相关过滤和最近上下文记录。

## 实施步骤

1. 新增 `answer_from_data()` 格式测试，先证明当前回答缺少“命中原因”和“可继续操作”。
2. 增强 `answer_from_matches()` 输出。
3. 运行 Knowledge、Agent 和全量验证。
4. 更新 README、进度文档、方案文档、验证记录和 `.codex` 本地记录。
