# 桌面快捷入口补齐最近上下文和最近文件计划

> 日期：2026-05-23  
> 执行者：Codex

## 目标

把“最近上下文”和“最近文件”补到桌面面板与托盘快捷命令中，让跨入口任务建议能力更容易触达。

## 范围

- `quick_commands()` 增加最近上下文和最近文件。
- `direct_quick_commands()` 将两者作为无参数快捷入口暴露。
- 面板和托盘无需单独改列表，继续复用 `direct_quick_commands()`。
- 继续排除 `/organize-preview` 这类需要参数的命令。

## 实现步骤

1. 更新桌面桥接、面板和托盘测试，先看到 RED。
2. 更新 `src/jarvis_lite/desktop/bridge.py` 的快捷命令集合。
3. 运行桌面桥接、桌面组件、托盘专项测试。
4. 运行全量测试、桌面 smoke 和 `git diff --check`。
5. 更新 README、验证记录和方案文档。

## 验收

- 桌面面板快捷按钮包含“最近上下文”和“最近文件”。
- 托盘快捷命令自动包含这两个入口。
- 点击“最近上下文”会提交 `查看最近上下文`。
- `/organize-preview` 仍不会进入直接快捷命令。
