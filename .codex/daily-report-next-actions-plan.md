# 日报下一步建议生成计划

> 日期：2026-05-22  
> 执行者：Codex

## 目标

让 `/daily-report` 在报告末尾生成确定性的下一步建议，帮助用户从最近上下文继续操作。

## 规则

- 最近资料存在：建议 `/read 最近资料` 和 `/tag 最近资料 标签...`。
- 最近目录存在：建议 `/organize-preview 别名` 和 `/dir-open 别名`。
- 最近建议存在：建议“查看第一条建议”和“执行第一条建议”。
- 有经验记忆：建议用 `/experience-advice 关键词` 复用经验。
- 有工具日志：建议用 `/experience 经验内容` 沉淀复用流程。
- 都没有时：给出导入资料、登记目录或记录经验的起步建议。

## 验证

1. 新增 Automation 测试，先证明日报缺少“下一步建议”段。
2. 实现确定性建议生成。
3. 运行 Automation、Agent、全量测试、桌面 smoke 和 `git diff --check`。
