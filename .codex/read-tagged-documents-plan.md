# 按标签读取资料组计划

> 日期：2026-05-25
> 执行者：Codex

## 目标

让用户在看到 `/kb-summary` 标签分组后，可以直接说“读取项目标签资料”，列出该标签下的资料，并继续按编号读取。

## 实现设计

- 在自然语言意图层新增 `read_tagged_documents`，标签文本放入 `alias` 字段。
- Agent 复用 `build_knowledge_index()` 按标签筛选资料。
- 输出标签资料列表、行数、标签和首条摘要预览。
- 匹配资料写入最近资料列表，后续“读取第一份资料”继续复用已有编号读取能力。
- 标签不存在时返回明确提示，不修改最近资料上下文。

## 验收标准

- Agent 测试覆盖成功读取标签资料组，并能继续按编号读取。
- Agent 测试覆盖标签不存在的提示。
- Agent、Knowledge、全量测试、桌面 smoke 和 `git diff --check` 通过。
