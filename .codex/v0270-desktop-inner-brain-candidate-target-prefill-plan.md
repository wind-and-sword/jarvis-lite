# 0.27.0 桌面候选目标预填计划

> 日期：2026-06-01
> 执行者：Codex

## 目标

让桌面候选训练区在选择候选后，可以显式预填常见教学命令或常见 `intent slot=value` 标注模板，降低人工训练输入成本。

## 边界

- 不改后端训练协议。
- 不新增自然语言正则。
- 不自动提交、不自动训练。
- 真实保存仍由用户检查输入框内容后手动发送。
