# 经验建议引用最近上下文第一版计划

> 日期：2026-05-22
> 执行者：Codex

## 目标

让 `/experience-advice 这个资料` 和 `/experience-advice 这个目录` 能引用当前最近上下文，输出具体对象和可直接复制调整的命令。

## 设计

- 最近资料：
  - 有最近资料时输出 `当前资料：data/...`。
  - 建议 `/read ...`、`/tag ... 标签...`、`/ask 问题`。
  - 搜索经验时把“这个资料”映射为“资料”，以便命中已有资料经验。
- 最近目录：
  - 有最近目录时输出 `当前目录：别名 -> 路径`。
  - 建议 `/organize-preview 别名` 和 `/dir-open 别名`。
  - 搜索经验时把“这个目录”映射为“目录”。
- 缺少上下文时输出明确提示，并保留通用命令建议。

## 非目标

- 不自动执行建议命令。
- 不推断文件路径或目录路径。
- 不改变经验存储、最近上下文持久化格式或自然语言大范围解析。
