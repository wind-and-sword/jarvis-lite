# 经验操作建议第一版计划

> 日期：2026-05-21
> 执行者：Codex

## 目标

让用户问“我该怎么导入资料”这类操作问题时，Jarvis Lite 能优先引用已有经验，并提示可继续使用 `/experience-search 关键词` 查看完整匹配。

## 范围

- 新增 `/experience-advice 关键词`。
- 新增自然语言映射：
  - “我该怎么导入资料”
  - “导入资料有什么经验”
  - “给我导入资料的建议”
- 复用 `search_experiences()`，不新增经验存储格式。
- 第一版只做关键词包含匹配，不做开放式计划、不调用大模型、不做评分或分类。

## 验收

- 有匹配经验时输出 `操作建议：关键词`、相关经验编号和后续命令提示。
- 无匹配经验时输出明确空结果，并提示先记录经验或使用 `/experience`。
- 缺少关键词时输出用法。
- 自然语言问法能映射到同一条建议路径。
- 全量 `unittest`、桌面 smoke 和 `git diff --check` 通过。

## 执行步骤

1. 写 Agent 失败测试，覆盖命令成功、空结果、缺关键词和自然语言映射。
2. 运行专项测试确认 RED。
3. 在 `agent.py` 增加 `/experience-advice` 和 `_experience_advice()`。
4. 在 `intent.py` 增加操作建议自然语言解析。
5. 运行专项 GREEN。
6. 更新 README、进度、验证和 `.codex` 记录。
7. 运行收尾验证并提交。
