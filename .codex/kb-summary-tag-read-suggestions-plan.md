# 知识库摘要提示按标签读取资料组计划

> 日期：2026-05-25
> 执行者：Codex

## 目标

在 `/kb-summary` 已经展示标签分组、按标签提问和自然语言按标签读取资料组之后，补充直接可用的“读取标签资料”建议，让用户从摘要页即可发现新入口。

## 实现设计

- 复用 `KnowledgeIndex` 中已有标签，不新增数据结构。
- 在 `_knowledge_summary()` 中，当知识库存在标签时追加一行 `按标签读取：读取标签A标签资料；读取标签B标签资料`。
- 标签排序和数量上限与现有 `/ask 标签` 建议保持一致，最多展示 3 个。
- 保持原有摘要正文、标签分组、最近资料列表写入和 `/ask` 建议不变。

## 验收标准

- Agent 测试先失败后通过，覆盖 tagged `/kb-summary` 输出按标签读取建议。
- Agent、Knowledge、全量测试、桌面 smoke 和 `git diff --check` 通过。
