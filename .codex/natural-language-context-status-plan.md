# 自然语言最近上下文状态计划

> 日期：2026-05-21
> 执行者：Codex

## 目标

让用户能用自然语言询问助手当前还记得哪些最近上下文，例如“查看最近上下文”“最近上下文状态”“你还记得刚才什么”。

## 范围

- 展示最近资料：`data/<relative_path>` 或无。
- 展示最近目录：`别名 -> 绝对路径` 或无。
- 展示最近搜索结果：结果数量和编号路径列表，或无。
- 复用现有运行态字段，不新增外部依赖。

## 非目标

- 不在本阶段持久化最近资料和最近目录。
- 不改变 `/ask` 检索排序、命中数量和回答格式。
- 不读取资料正文；查看正文继续使用“查看第二条结果”。

## 验收

- 没有任何上下文时，自然语言查询返回明确空状态和下一步提示。
- 问答命中多条资料并打开目录后，自然语言查询能列出最近资料、最近目录和最近搜索结果。
- 新 Agent 实例加载已持久化的最近搜索结果后，状态查询能展示最近搜索结果。
- `tests.test_agent` 专项通过，全量 `unittest discover`、桌面 smoke 和 `git diff --check` 通过。
